## Features

- Claude Code automation

## Getting Started

Clone this repository and install dependencies:

```bash
git clone https://github.com/drkreddy/sample.git
cd sample
npm install
```

### Running the Application

Start the application:

```bash
npm start
```

For development with auto-reload:

```bash
npm run dev
```
